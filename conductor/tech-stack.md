# Tech Stack

## Core
- **Language:** Ruby 4.0.1
- **Framework:** Ruby on Rails 8.1
- **Database:** PostgreSQL

## Frontend
- **Views:** Phlex (RubyUI)
- **Interactivity:** Hotwire (Turbo + Stimulus)
- **Styling:** Tailwind CSS

## Testing & Quality
- **Unit/Integration Tests:** RSpec
- **System/E2E Tests:** Capybara, Playwright
- **Test Data:** Rails Fixtures

## Infrastructure & Tooling
- **Observability:** OpenTelemetry
- **Environment Management:** Mise
- **Task Runner:** Taskfile
